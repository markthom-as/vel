# 40-01 Plan

## Slice

Create the cross-surface discovery baseline for Phase 40: what works, what is broken, and what is merely confusing across web and mobile.

## Work

- audit `Now`, `Threads`, `Settings`, and context/right-panel behavior on web
- audit primary iPhone/mobile daily-use interactions that overlap the same surface model
- classify findings into:
  - working
  - broken/inert/misrouted
  - hierarchy/clarity debt
- write a durable audit artifact that later slices can close against

## Verification

- audit artifact exists and cites the actual shipped surfaces
- targeted code/doc references prove the audit is grounded in current implementation

## Exit

Phase 40 has a factual baseline instead of relying on assumed UI complaints.
