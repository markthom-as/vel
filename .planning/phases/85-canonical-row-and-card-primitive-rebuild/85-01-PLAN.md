# Phase 85 Plan

## Objective

Collapse the current shared primitive drift into one canonical row skeleton and a small role-based card taxonomy.

## Scope

### In scope

- introduce a canonical row primitive in `clients/web/src/core/`
- introduce a bounded card primitive in `clients/web/src/core/`
- route existing `NowItemRow`, `PanelChrome`, and generic card layouts through the new primitives
- reduce metric-strip visual prominence without breaking existing surface contracts

### Out of scope

- full page rewrites for `Now`, `Threads`, or `System`
- drawer redesign
- browser proofs

## Implementation slice

1. Add `ObjectRow` as the base row skeleton:
   - leading slot
   - content block
   - optional subtitle
   - bounded metadata/tags area
   - trailing action zone

2. Add `ObjectCard` as the bounded card taxonomy:
   - default
   - nudge
   - run
   - artifact
   - config

3. Convert compatibility wrappers:
   - `NowItemRow*` wraps `ObjectRow`
   - `Panel*` section/inset/dense-row wrappers wrap `ObjectCard` / `ObjectRow`
   - generic `CardLayout` wraps `ObjectCard`

4. Tone down metric strips so they remain supporting rather than dominant.

## Verification

- targeted web tests around touched shell/surface files
- production web build
- primitive compatibility holds for existing views without page-specific rewrites
