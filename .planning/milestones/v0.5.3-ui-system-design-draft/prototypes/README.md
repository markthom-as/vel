# Phase 82 Prototypes

These HTML/CSS prototypes are the primary mockup artifacts for Phase 82.

They exist to validate:

- shell law
- disclosure maps
- bounded `Now`
- object/context-first `Threads`
- hybrid read-first / operational `System`

## Included states

- `now-normal.html`
- `now-degraded.html`
- `thread-normal.html`
- `thread-focused.html`
- `system-integrations-issue.html`
- `system-control.html`

## Notes

- desktop-first
- interaction fidelity over polish
- static HTML/CSS with light JS affordances
- intended for in-repo review and implementation handoff
