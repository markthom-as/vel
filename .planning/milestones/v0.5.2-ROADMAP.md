# Milestone v0.5.2: Operator Surface Embodiment

**Status:** SHIPPED 2026-03-23
**Phases:** 72-77
**Total Plans:** 6

## Overview

`v0.5.2` turned the truthful `0.5.1` client line into the intended operator UI state on web without reopening backend law.

This milestone intentionally excluded backend renegotiation, framework migration, new providers, workflow-builder widening, and Apple implementation. It closed by embodying the three web surfaces, proving them in the browser, and refreshing Apple handoff guidance around the embodied surface model.

## Accomplishments

- Phase `72` froze the operator-surface doctrine and the approved UI contract before implementation spread.
- Phase `73` rebuilt the shared shell rhythm, removed the global info rail, and established the reusable disclosure substrate.
- Phase `74` turned `Now` into the intended focus-first operator surface and repaired active-path completion latency.
- Phase `75` made `Threads` object-grounded and chronology-secondary with browser-proven bound-object reading.
- Phase `76` turned `/system` into a grouped sidebar/detail structural surface while preserving the frozen action allow-list.
- Phase `77` closed the line with cross-surface proof, milestone evidence, and refreshed Apple handoff docs.
