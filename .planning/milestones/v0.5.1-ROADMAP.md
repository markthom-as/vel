# Milestone v0.5.1: Canonical Client Reconnection

**Status:** SHIPPED 2026-03-23
**Phases:** 66-71
**Total Plans:** 6

## Overview

`v0.5.1` reconnected the web client to the frozen `0.5` backend through the smallest truthful surface set: `Now`, `Threads`, and `System`.

This milestone intentionally excluded backend renegotiation, framework migration, broad UI redesign, new providers, and Apple implementation. It closed by proving the shipped truth in the browser, deleting or quarantining stale seams, and publishing an Apple handoff packet.

## Accomplishments

- Phase `66` froze the truthful-surface doctrine, three-surface model, and canonical client boundary.
- Phase `67` turned stale client/backend touchpoints into an explicit kill list with rewrite, delete, and quarantine dispositions.
- Phase `68` landed one typed canonical transport layer and removed direct fetch escapes from the active client path.
- Phase `69` rebound `Now` to canonical task/calendar truth with adjacent sections and direct canonical completion flow.
- Phase `70` collapsed the shell to `Now`, `Threads`, and `System`, removed `Settings` / `Projects`, and shipped the minimal authoritative `/system` surface.
- Phase `71` added browser-executed proof, documented bounded reads/quarantines, and produced the Apple handoff packet.
