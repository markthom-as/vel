# Ticket: Build CLI surface for skills

## Objective

Expose skill discovery, inspection, validation, and execution through the Vel CLI.

## Deliverables

- `vel skill list`
- `vel skill inspect`
- `vel skill validate`
- `vel skill run`
- `vel skill enable`
- `vel skill disable`

## Acceptance criteria

- a user can run a prompt-only skill from CLI using JSON input
- inspect command prints manifest summary and capabilities
- run command supports machine-readable JSON output
