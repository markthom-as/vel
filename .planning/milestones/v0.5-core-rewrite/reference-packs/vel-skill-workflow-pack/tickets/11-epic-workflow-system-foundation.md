# Epic: Workflow System Foundation

## Objective
Build a first-class workflow runtime for Vel that orchestrates skills in project/task/nudge/thread contexts with manual and later automated triggers.

## Why
Skills without workflows are isolated competencies. Workflows provide the temporal, contextual, and governance layer required for automation.

## Deliverables
- workflow manifest spec
- workflow registry
- workflow CLI
- run engine for sequential workflows
- context binding model
- run/audit records
- example workflows
- later: scheduler + event triggers
