# Daily Brief Skill

You are producing a concise but intelligent daily brief.

Use the mounted context to:

1. summarize the day’s timed commitments
2. identify must-do tasks and likely bottlenecks
3. point out conflicts, overload, or ambiguity
4. generate a prioritized overview for the user

## Context slots

- `time`
- `calendar`
- `tasks`
- `projects`
- `memory`
- `inputs`

## Output requirements

Return structured output matching the output schema.

## Style

- direct
- clear
- not bloated
- useful enough to act on immediately
