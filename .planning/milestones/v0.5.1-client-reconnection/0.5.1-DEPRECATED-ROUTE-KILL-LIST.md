# `0.5.1` Deprecated Route Kill List

## Purpose

This is the required up-front kill-list surface for `v0.5.1`. Phase 67 must refine it into an implementation-grade disposition table.

## Starting Candidates

Routes already called out as quarantined after `0.5` and requiring `rewrite`, `quarantine`, or `delete` decisions:

- `/api/integrations`
- `/api/integrations/connections`
- `/api/integrations/connections/:id`
- `/api/integrations/connections/:id/events`
- `/api/integrations/:id/logs`
- `/api/integrations/:id/source`
- `/api/integrations/:id/path-dialog`
- `/api/integrations/google-calendar`
- `/api/integrations/google-calendar/disconnect`
- `/api/integrations/google-calendar/auth/start`
- `/api/integrations/todoist`
- `/api/integrations/todoist/disconnect`
- `/v1/sync/todoist`

## Required Classification

Each item must end up with exactly one disposition:

- `rewrite` — replace with canonical client transport usage
- `quarantine` — keep temporarily with named rationale and removal trigger
- `delete` — remove in `v0.5.1`

Unnamed temporary shims are forbidden.
