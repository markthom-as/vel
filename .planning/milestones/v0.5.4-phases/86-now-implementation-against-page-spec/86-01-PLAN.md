# Phase 86 Plan

## Objective

Rebuild `Now` into the bounded execution-first surface defined by the locked page spec.

## Scope

### In scope

- reduce `Now` to:
  - active task
  - one next slot
  - current/next event constraints
  - optional degraded trust state
  - transient completion acknowledgement
- keep nudges owned by the shell-side nudge zone
- preserve minimal inline actions for task completion and thread escalation

### Out of scope

- inbox-style queues
- triage dashboards
- project grouping
- deeper drawer work

## Implementation slice

1. Remove multi-section queue/triage/dashboard structure from `NowView`.
2. Rebuild `NowView` around bounded cards/rows from the new primitive layer.
3. Move nudge-route behavior assertions to the shell-owned `NudgeZone`.
4. Preserve completion reconciliation and degraded trust routing.

## Verification

- targeted `Now`, `NudgeZone`, `MainPanel`, `Threads`, and `System` tests
- production web build
