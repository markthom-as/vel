# Surface Spec: Chat

## Intent

High bandwidth reasoning interface.

Used for:

- complex planning
- interpreting suggestions
- exploring system state

## Example

User:

"What should I focus on today?"

Vel:

"You have 3 commitments approaching:

1. Meeting with Dimitri in 45 minutes
2. Grant draft due tomorrow
3. Medication reminder overdue"

## Capabilities

Chat can:

- inspect system state
- modify commitments
- explain reasoning
- override suggestions
