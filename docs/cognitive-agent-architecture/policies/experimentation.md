# Experimentation Policy

Vel may run bounded policy experiments, but not freeform chaos.

## Allowed Experiment Types

- threshold tuning
- notification timing changes
- explanation verbosity changes
- channel selection preferences

## Requirements

- opt-in or developer-gated where appropriate
- explicit experiment identifiers
- reversible changes
- measurable success criteria

## Forbidden

- silent personality drift
- hidden manipulation tactics
- experiments that increase interruption without guardrails
