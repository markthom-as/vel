# Evaluation Metrics

Vel needs system metrics that map to user value.

## Core Metrics

- reminder acknowledgment latency
- commitment completion rate
- suggestion acceptance rate
- false-positive risk alert rate
- duplicate notification rate
- escalation success rate between surfaces
- user override frequency

## Reflection Metrics

- policy change win rate
- regression detection rate
- experiment rollback frequency

## Warning

Do not optimize only for clicks or acknowledgments.
A system can become very "engaging" while being profoundly annoying.
