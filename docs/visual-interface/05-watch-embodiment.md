# Watch Embodiment

The watch embodiment is not a tiny desktop port.

It is the condensed glyph of the same personality field.

## Basis states

Implement at least these authored basis states:

1. `idle_diffuse`
2. `idle_coherent`
3. `attentive`
4. `thinking_split`
5. `speaking_warm`
6. `warning_tense`
7. `fragmenting`
8. `sleeping`

## Runtime behavior

The watch renderer should:
- choose the nearest 2–3 basis states
- blend them by weight
- apply live modifiers:
  - pulse rate
  - hue bias
  - asymmetry offset
  - turbulence amplitude
  - faciality gain

## Visual priorities

Prioritize:
- silhouette
- mouth seam
- brow pressure band
- pulse cadence
- single-vs-many lobe logic
- hue drift

Deprioritize:
- high-frequency fluid detail
- many satellites
- heavy transparency
- expensive post FX
