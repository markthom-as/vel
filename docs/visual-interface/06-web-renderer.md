# Web Renderer Reference

The web renderer is the reference embodiment for desktop and mobile.

## Recommended stack

- React
- TypeScript
- React Three Fiber or Three.js
- shader/SDF metaball approach
- lightweight controls/debug UI

## Required visual features

- primary cranial mass
- secondary orbiting/merging masses
- implied nose bridge
- mouth seam
- brow pressure shelf
- no eyes
- turbulence mapped to affect state
- color mapped to affect state
- merge/split behavior mapped to coherence/fragmentation/sociality

## Mobile policy

Mobile can use the same renderer with:
- fewer satellites
- simplified noise
- adaptive resolution
- frame throttling when idle

This is a quality reduction, not a conceptual reduction.
