# Vel Uncertainty Pack

This pack operationalizes uncertainty as a first-class runtime concept.

Vel already contains scattered confidence notions:
- `confidence` in chat schema
- `confidence` in inferred state rows
- attention confidence in current-context JSON
- uncertainty-oriented specs in `docs/specs/vel-uncertainty-architecture-spec.md`

What is missing is a unified subsystem.
